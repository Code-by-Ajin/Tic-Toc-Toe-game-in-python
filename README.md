Before run this code,You will need to install the Pygame library. Open your terminal or command prompt and run:

pip install pygame


Before running the code, create a folder named music in the same directory as the python script. Add 5 audio files named 1.mp3, 2.mp3, 3.mp3, 4.mp3, and 5.mp3. (If you don't add them, the game will still run perfectly, just without music).
